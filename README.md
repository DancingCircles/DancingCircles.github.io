## 📂 项目列表

### [XOBERON - 个人作品集网站](./XOBERON)

一个现代化的交互式个人作品集，展示创意设计和前端开发技能。

**主要特性：**
- 🎨 复古加载动画（1950-2025年代计数器）
- 🌊 流体波浪效果（OGL交互式动画）
- 🎬 滚动驱动动画（GSAP ScrollTrigger）
- 🎯 3D元素（Three.js驱动）
- 💫 自定义光标
- 📱 完全响应式设计

**技术栈：**
- Vite 5.x + React 18.x
- Three.js + React Three Fiber
- GSAP + Lenis
- Framer Motion
